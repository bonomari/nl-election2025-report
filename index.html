🗳 現在の政治情勢（2025年7月末時点）

2023年のルッテ政権崩壊以降、オランダ政治は大きく変動。2024年末の下院選では、ヘールト・ウィルダース（Geert Wilders）率いる極右政党PVV（自由党）が第1党となり、現在はNSC（新社会契約党）、VVD（自由民主党）などと連立協議を進めたが、最終的には外部協力型内閣として、右派中心の政権構造が固まりつつある。

  • 現在の政治対立の焦点は以下：
    • ✅ NATO国防費のGDP5%案
    • ✅ 社会保障の削減・増税策（窒素税を含む）
    • ✅ 移民政策と国籍法改正案
    • ✅ 気候政策と農業利害（BBBの影響を含む）
    • ✅ 政治家のX（旧Twitter）発言による対立激化

🗓 2025年 総選挙に向けた主な政治動向（〜7月末）

📌 2024年12月

  • 総選挙の結果発表
    極右政党PVV（ヘールト・ウィルダース / Geert Wilders）が第1党に。中道のNSC（ピーテル・オムツィフト / Pieter Omtzigt）、VVD（ディラン・イェシルギュズ / Dilan Yeşilgöz）、GL–PvdA（ロブ・イェッテン / Rob Jetten）などと連立協議が始まる。

📌 2025年1月〜3月

  • 新内閣構想、断続的な協議
    PVV主導の連立交渉は難航。NSCとGL–PvdAの参加に壁。結局、PVV＋VVDの協調体制＋NSCの外部支援という構造へ移行。

📌 2025年4月

  • 防衛支出増案の浮上（NATO関連）
    ヘールト・ウィルダースがXで「国防は最大の福祉」と発言、NATO支出をGDPの5%まで引き上げる案を推進。
    • 引用元：https://twitter.com/geertwilderspvv

📌 2025年5月

  • 社会保障圧縮・移民制限の論争
    NSCが「バランスある財政構造を目指す」と表明、GL–PvdAが「軍拡と福祉切り捨ては矛盾」と反論。

    • 出典：Volkskrant

  • 窒素税（stikstofbelasting）の導入が議論に浮上
    GL–PvdAが農業圧迫を伴う形でも環境対策推進へ。VVDは「企業と農家のバランスが必要」と慎重姿勢。

📌 2025年6月

  • NSC：選挙マニフェスト草案を公開（非公式）
    オムツィフトが「財政規律と社会的公正の両立」と主張。気候・農業・防衛すべてを漸進改革で対応。

    • 出典：Telegraaf

  • BoerBurgerBeweging（BBB）の存在感低下
    窒素税問題で主張を強めるも、PVVとNSCが保守層の吸収を強化。支持率は昨年のピーク時から半減。

📌 2025年7月

  • NATO支出支持、国民の半数超が賛成
    NLTimes報道によると、「GDP5%支出」に56%が賛成と回答（I&O調査）

    • 出典：https://nltimes.nl/2025/07/15/majority-dutch-support-boosted-defense-spending

  • GL–PvdA、ロブ・イェッテンが「EUとのバランス重視」を強調
    NATO支出案を「偏った安全保障戦略」と批判

  • ウィルダース、Xで再度挑発的投稿
    「我々はEUの命令ではなく、オランダ人の安全を優先する」

🔚 次のステップ（8月以降）

  • 選挙マニフェスト正式発表、TV討論、候補者リスト公開が本格化
  • 各党のウェブサイトおよびSNS発信が主戦場となる
